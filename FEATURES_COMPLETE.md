## 🚀 HOW TO USE NEW FEATURES

### 1. Manage Cards
1. Click **🎴 Cards**
2. Select card type (Debit/Credit)
3. Choose linked account
4. Click **Request Card**
5. Card appears in your list

### 2. Invest Money
1. Click **📈 Investments**
2. Select investment type
3. Enter investment amount
4. Set tenure (months)
5. Enter expected return %
6. Click **Start Investment**

### 3. Pay Bills
1. Click **📄 Bills**
2. Select bill type
3. Choose source account
4. Enter bill amount
5. Add reference number
6. Click **Pay Now**

### 4. Create Savings Goals
1. Click **🎯 Goals**
2. Enter goal name
3. Set target amount
4. Pick target date
5. Set monthly contribution
6. Click **Create Goal**
7. Track progress with visual bar

### 5. Add Beneficiaries
1. Click **👥 Beneficiaries**
2. Enter beneficiary name
3. Add account number
4. Enter bank name
5. Add IFSC code
6. Select relationship
7. Click **Add Beneficiary**
